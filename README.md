# Document Framework

Mintlify documentation scaffold for a reusable documentation framework.

## Local development

```bash
npm i -g mint
npm run dev
```

Open `http://localhost:3000`.

This workspace uses the installed Node `v24.15.0` from `~/.nvm` because Mintlify does not support Node 25+.

## Files

- `docs.json` configures Mintlify theme, colors, navbar, and sidebar navigation.
- `index.mdx` is the docs homepage.
- `document-framework.mdx` is the main documentation skeleton.
- `reference/template.mdx` is a reusable page template.
